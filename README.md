

# 📚 Library Management System (C)

A simple console-based Library Management System built using C programming.

## 🚀 Features
- 👨‍🎓 Student Login
  - View all books
  - Search book by ID or Title

- 👨‍💼 Librarian Login
  - Add new books
  - View all books
  - Search books
  - Update book details
  - Delete books

## 🛠️ Technologies Used
- C Programming Language
- Standard Libraries (`stdio.h`, `string.h`)

## ▶️ How to Run

1. Compile the program:
   gcc library_management.c -o library

2. Run the program:
   ./library
